# DADDY'S HOME - Luxury Architecture & Design

A world-class premium luxury website specializing in Interior Design, Exterior Design, Garden Design, 3D Home Visualization, and Home Customization Services. Built to deliver a high-end, interactive user experience.

## ✨ Features

- **Premium Design System:** Matte Black, Deep Charcoal, Warm Beige, Ivory White, and Gold color palette with glassmorphism UI elements.
- **Interactive 3D Experience:** A fully integrated 3D luxury home model that users can rotate and zoom using `Three.js`.
- **Advanced Animations:** Smooth page transitions, scroll-triggered reveals, and a custom magnetic cursor powered by `GSAP`.
- **AI Customization Mockup:** A drag-and-drop file upload feature that simulates an AI design transformation.
- **Dynamic Portfolio:** A filterable masonry layout showcasing luxury design projects.
- **Dark/Light Mode:** Seamless theme switching to accommodate user preferences.

## 🛠️ Technologies Used

- **Vite:** Next-generation frontend tooling for ultra-fast development and building.
- **Vanilla JavaScript:** Clean, framework-agnostic logic for maximum performance.
- **Three.js:** For rendering the interactive 3D WebGL scene.
- **GSAP (GreenSock):** For high-performance, professional-grade animations.
- **HTML5 & CSS3:** Semantic markup and modern CSS variables/flexbox/grid layouts.

## 🚀 Running Locally

1. **Install Dependencies:**
   Make sure you have Node.js installed, then run:
   ```bash
   npm install
   ```

2. **Start the Development Server:**
   ```bash
   npm run dev
   ```

3. **View in Browser:**
   Open [http://localhost:5173](http://localhost:5173) to view the site.

## 🌍 Deployment (Vercel)

This project is configured for easy deployment on **Vercel**. 

1. Upload all source files to a GitHub repository (do **not** upload the `node_modules` or `dist` folders).
2. Import the repository into your Vercel dashboard.
3. Vercel will automatically detect the Vite framework and use the included `vercel.json` file for routing.
4. Click **Deploy**!

*Build Command:* `npm run build`  
*Output Directory:* `dist`
